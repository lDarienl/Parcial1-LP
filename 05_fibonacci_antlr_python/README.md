# Punto 5 — ANTLR (Python) para `FIBO(n)`

Entrada por consola:

```
FIBO(20)
```

Salida por consola (Fibonacci **hasta n**, es decir valores \(\le n\)):

```
0, 1, 1, 2, 3, 5, 8, 13
```
## Ejecutar

Interactivo:

```bash
python3 main.py
```

O pasando el input como argumento:

```bash
python3 main.py "FIBO(20)"
```
