# Parcial 1 — Lenguajes de Programación

Repositorio con las soluciones del **Parcial 1**. Cada punto vive en una carpeta independiente con su propio `README.md`, código fuente y ejemplos de entrada.

## Estructura

- `01_afd_ajedrez_python/`: Expresión regular de movimientos (tipo `p->k4`, `kbp X qn`) + implementación de **AFD** en Python.
- `02_afd_id_python/`: AFD en Python para IDs con la ER: \([A-Za-z][A-Za-z0-9]*\) + pruebas.
- `03_calc_sqrt_flex_bison/`: Calculadora en C con **Flex + Bison** que calcula `sqrt(x)` usando **Newton-Raphson**. Entrada por archivo, salida por consola.
- `04_rendimiento_compilado_vs_interpretado/`: Comparación de rendimiento (compilado vs interpretado) usando una función recursiva.
- `05_fibonacci_antlr_python/`: ANTLR (target Python) para evaluar `FIBO(n)` e imprimir la secuencia de Fibonacci **hasta \(n\)**.
